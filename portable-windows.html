Portable LibreOffice for Windows — Run from a USB Drive

Use LibreOffice without installing it on the PC. A portable edition keeps settings in its own folder and can travel with your USB drive. For best performance and integration on a single machine, we recommend the full MSI install.

Prefer a full install? Get the official MSI (64‑bit): LibreOffice_26.2.2_Win_x86-64.msi • Size: —
Download MSI (64‑bit) Learn about portable versions
How to use a portable edition
  1. Download a trusted portable build from official sources.
  2. Extract the folder to your USB drive (e.g., D:\\LibreOfficePortable\\).
  3. Run soffice.exe from the portable folder. Settings and extensions stay within the folder.
  4. Close LibreOffice before safely ejecting the USB drive.

Pros and cons

Pros
  • ✓ No installation or admin rights required
  • ✓ Keep your settings, templates and extensions with you
  • ✓ Ideal for shared or locked‑down PCs
Cons
  • ✕ Slower startup on USB drives
  • ✕ No system‑wide file associations by default
  • ✕ Manual updates required

Good to know

  • Portable builds store profiles locally; back up the folder to keep settings.
  • To open files by double‑click, use the full MSI install or set associations manually.
  • On very slow USB drives, copy large files to the PC first for better performance.

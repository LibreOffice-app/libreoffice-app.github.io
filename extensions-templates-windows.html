How to install extensions (.oxt)

  1. Open LibreOffice on Windows.
  2. Go to Tools → Extension Manager → Add…
  3. Select the .oxt file → Accept the license → Restart LibreOffice.
  4. Manage or disable add‑ons later in Extension Manager.

How to import templates

  1. Open LibreOffice → File → Templates → Manage.
  2. Click Import → pick your .ott/.ots/.otp file.
  3. Assign a category and confirm. Your template appears in the gallery.

Windows quick notes

  • Extensions work per‑user by default; install for all users via admin if needed.
  • After updates, re‑enable or update extensions if they were disabled.
  • Templates sync well with cloud folders (OneDrive, etc.) if stored in Documents.

How to Install LibreOffice on Windows 11/10 (MSI, 64-bit)

This guide shows how to install LibreOffice 26.2.2 using the official MSI offline installer. It works on Windows 10 and Windows 11 (64‑bit).

1

Download the MSI installer

Click the button above to download LibreOffice_26.2.2_Win_x86-64.msi from the official LibreOffice servers.

If your browser asks what to do with the file, choose Save.

2

Run the installer

Open your Downloads folder and double‑click the MSI file. If Windows shows a security prompt (UAC or SmartScreen), click Run or Yes.

3

Choose setup type

For most users, pick Typical. Advanced users can choose Custom to select components and languages. You can also decide whether to create Start Menu and desktop shortcuts.

4

Install and wait

Click Install. Close any open Microsoft Office or LibreOffice apps if requested. The copy process usually takes a few minutes.

5

Finish and launch

Click Finish. Start LibreOffice from the Start menu. Open Writer, Calc and Impress to verify everything works.

Optional: set default app associations for .odt/.ods/.odp files in Windows Settings.

FAQ

  • Is this an offline installer? Yes — the MSI is a full, offline package.
  • Does it work on Windows 11? Yes — Windows 10 and Windows 11 (64‑bit) are supported.
  • I got an error during install. What now? See Troubleshooting: Fix installation errors.
